== Description ==

The TMR mortgage calculator widget is a free 3-in-1 calculator allowing users to calculate totals costs which include taxes, insurance, PMI, and HOA fees. The calculator is configurable with 11 color schemes and 2 sizes making it easy to integrate into your sidebar or inside the body of pages or posts.

Calculate:

- Total monthly payments
- How much home you can afford
- Purchase price based on a set monthly cost

== Installation ==

1. Upload the `tmr_mortgage_calculator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Frequently Asked Questions ==

= Is there any cost or paid features for the plugin? =

The calculator widget is completely free, with no need to pay to unlock any features.

= What about the amortization table? =

In addition to the standard calculations. you can also view a detailed report which includes a full amortization schedule for you to print out and save.
